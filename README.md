# Book Donation

Check out current hosting here: https://bookd-web.herokuapp.com

## Installation

1. Clone the repository

   ```sh
   git clone https://github.com/CSEKINGS/book-donation-react-ts-web.git
   ```

2. Install NPM packages

   ```sh
   npm install
   ```

3. Run

   ```sh
   npx vite
   ```
